== Description ==

The *Tortuga Pro* plugin adds additional features to the Tortuga theme.

= Features =
* Theme Color Settings
* Typography Options
* Top & Footer Navigation
* Header Search feature
* Social Icons Menus
* Additional Widget Areas
* Author Bio
* Wigetized Footer Area
* Scroll-to-Top Button

== Installation ==

1. Install the Tortuga Pro Plugin on Plugins -> Add New -> Upload Plugin by uploading the .ZIP plugin file.
1. Enter your License Key on Appearance -> Pro Version to receive updates and support.
1. Enjoy the new features.

Please also see our [Plugin Installation Tutorial](https://themezee.com/docs/install-wordpress-plugins/ "Plugin Installation Tutorial").

== Frequently Asked Questions ==

= Where do I get help to configure this plugin?  =
Please see our [Tortuga Theme Documentation](https://themezee.com/docs/tortuga-documentation/ "Tortuga Theme Documentation").

Please go to *Appearance -> Customize -> Theme Options -> Pro Version* and enter your license key in order to receive updates.
